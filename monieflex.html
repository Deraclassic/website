MonieFlex

A web-based Fintech Application

Features

  • Transfer of funds
  • Airtime recharge
  • Bill payments
  • Card management
  • Security/Signup/Login

Technologies Used

  • Backend: Java, Spring Boot
  • Frontend: Reactjs, TailwindCSS
  • Database: PostgreSQL
  • Authentication: JSON Web Tokens (JWT)
  • Deployment: Docker, CI/CD
  • Testing: JUnit, Mockito
  • Email Notifications: JavaMailSender
  • API Documentation: Swagger (OpenAPI)
  • Email Template Engine: Thymeleaf

About Project

A web-based Fintech application for managing financial transactions. Features include sending/receiving money, debit card management, utility payments, and airtime/data purchase.
